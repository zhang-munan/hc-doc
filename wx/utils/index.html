# 表单修改值

在微信小程序更新数据时，普通的属性的绑定是单向的，简易的双向绑定 (opens new window)也是需要满足一定条件，不是适用发部分使用场景，所以项目中做了共同处理，减少代码量。

# input 框

<input value="{{form.name}}" bind:input="$handleInput" data-key="form.name" />
